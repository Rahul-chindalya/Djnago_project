SHIP MANAGEMENT PROJECT 

This project is a backend API system built using Django and Django REST Framework.
It is designed to manage users and simulate a Ship Maintenance System.
The accounts module handles user registration, login, and profile management.
JWT authentication is implemented for secure access using access and refresh tokens.
Role-based permissions are applied where Admin (superuser) can perform CRUD operations.
Inspectors and Engineers have view-only access to the data.
The project follows a clean structure with models, serializers, views, and URLs
